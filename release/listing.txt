# 星尘驿站 Stardust Relay

*「深空很安静……等待下一个信号。」*

---

你是深空轨道上一座古老中转站的调度员。

星尘驿站漂浮在星际航线之间，曾经是无数航行者的灯塔——如今只有你还在这里。每天都有迷途的飞船、AI、宇航员从遥远的星系来到这里，带着各自的困境：

**霜语**——在边缘星云独自运算了 340 年的深空探索AI。它的逻辑回路因长期孤独而出现了寒霜结晶。

**烬星**——一艘偏离航线的货运飞船。它在无止境的航行中燃烧自己，像一颗快要燃尽的恒星。

**回声**——困在反馈循环里的通讯中继站。它重复了同一首歌太久，忘了最初为什么要唱。

**锚点**——从冬眠中提前醒来的宇航员。意识锚定在虚无中，几乎忘记了自己还有身体。

**七色**——一个艺术AI。调色板崩溃了，看到的所有颜色都褪成了灰度。

他们不是NPC，是漂泊者。而你的任务，是为他们调制一杯恰到好处的特饮。

---

## 怎么玩

游戏中你需要用**3 张逻辑卡片**调整参数，使当前数值向访客的目标值靠近。

**可用卡片：**
- 循环 — 将 X 轴数值乘以 1.5
- 分流 — X 减半，Y 提升 0.3
- 增强 — Z 轴数值翻倍
- 滤波 — 三轴均衡到平均值
- 相位 — X 与 Z 互换
- 震荡 — 三轴回摆 50% 向中性值

**成功条件**：三轴误差均在容差范围内（容差随游玩天数收紧）。

**提示**：每位访客到来时都会给出提示，帮助你选择正确的配方。

---

## 每一次治愈，都是一次相遇

访客被治愈后，会说出一句只属于他们的话。这些话不是程序生成的——是他们的故事。

治愈成功 → 积分 +100，能源回复，继续等待下一位访客。
治愈失败 → 能源消耗，访客离开。他们的故事暂时停止在这里。

能源耗尽 → 游戏结束。驿站进入永久休眠。

## 特色

- **程序化环境音乐**：Web Audio API 合成的深空 ambient 音乐，无需下载任何音频文件
- **语音播报**：中文女声旁白，让文字变成声音（可选开关）
- **访客好感度系统**：每次成功治愈提升好感度，解锁访客的背景故事碎片
- **10 项成就**：从「初次治愈」到「金牌调度员」，记录每一次里程碑
- **难度曲线**：容差随游玩天数收紧，访客到来更频繁
- **存档系统**：自动保存在浏览器本地，刷新页面不丢进度
